# PHP Docker Dev Setup

This project now uses PHP includes for shared layout parts and can run in Docker.

## Start

```bash
docker compose up --build
```

Open `http://localhost:8081`.

## Base path (optional)

If you serve the site under a subdirectory, set `APP_BASE_PATH` in
`docker-compose.yml` (example: `/Sanobudokai_Official_Homepage`).
